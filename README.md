# Student Result Analysis Dashboard

An interactive student performance analysis dashboard built with Python and Streamlit that visualizes marks, attendance and results automatically.

## Features
- Student data table with all marks
- Pass/Fail pie chart
- Subject wise average marks bar graph
- Top 5 students ranking
- Attendance vs Marks scatter plot

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- OpenPyXL

## How to Run

Install dependencies:
pip install streamlit pandas matplotlib openpyxl

Create sample data:
python create_data.py

Run the dashboard:
streamlit run app.py
